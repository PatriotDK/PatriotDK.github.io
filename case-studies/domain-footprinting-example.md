# Domain Footprinting Example

## Objective
Map the public-facing footprint of a domain using passive OSINT methods.

## Scope
Target: example.com  
Method: passive, public-source only

## Tools Used
- whois
- dig
- theHarvester
- SpiderFoot
- Amass

## Workflow
1. Identify registrar / ownership indicators
2. Review DNS records
3. Collect subdomains
4. Search for exposed email patterns
5. Cross-check findings

## Output Categories
- Domain metadata
- DNS information
- Subdomains
- Email exposure
- Public technology indicators

## Key Lesson
The value of OSINT often comes from correlation, not just collection.
