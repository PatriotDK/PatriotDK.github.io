# SpiderFoot

## Purpose
SpiderFoot automates OSINT collection across many public data sources.

## Useful For
- domain profiling
- IP and subnet analysis
- email footprinting
- organization mapping
- broad reconnaissance

## Example Command

```bash
spiderfoot -l 127.0.0.1:5001
```

Then open the web interface and create a scan for:
- domain
- IP
- ASN
- email
- name

## Notes
- Good for broad discovery
- Results should be filtered for relevance
- Helpful for showing workflow screenshots in a portfolio
