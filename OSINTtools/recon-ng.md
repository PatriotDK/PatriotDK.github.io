# Recon-ng

## Purpose
Recon-ng is a modular web reconnaissance framework.

## Strengths
- workspace-based workflow
- modular collection
- database-backed results
- useful for repeatable investigations

## Example Workflow

```bash
recon-ng
marketplace search
marketplace install all
workspaces create demo
modules search
```

- Good for structured engagements
