# Cybersecurity Governance

## Overview

This section documents notes and structured knowledge related to **cybersecurity governance, compliance, and regulatory frameworks**.

The focus is on understanding how organizations implement cybersecurity requirements in practice, especially where **technology, law, and risk management intersect**.

---

## Topics Covered

This repository contains notes related to:

- Cybersecurity governance frameworks
- Risk management processes
- Regulatory compliance
- Security documentation
- Incident response governance
- Third-party and outsourcing risk

---

## Regulatory Frameworks

Particular focus is given to European cybersecurity regulation, including:

- NIS2 Directive
- DORA (Digital Operational Resilience Act)
- Cybersecurity Act
- Cyber Resilience Act
- GDPR security obligations

---

## Objectives

The purpose of this repository is to:

- Structure knowledge about cybersecurity governance
- Translate regulatory requirements into operational practices
- Explore how organizations implement compliance frameworks
- Connect legal requirements with technical security realities
